### Birthday Buddy App

This project is meant to showcase off the basics of React and how to use state to dynamically render components.

Main Concepts used:

- Vanilla JavaScript (contains concepts like destructuring objects, the .map function, the spread operator, etc...)
- ES6 Modules
- React Components as individual, standalone pieces of UI
- State, and the use of the useState hook
- Dynamic use of data

Hooks used:

- useState

Prerequisite knowledge required:

- Vanilla JavaScript (especially with regards with handling arrays)
- Basic level JSX
- Component Usage
- State

Technology required:

- JavaScript
- CSS
- HTML
- React
- Vite (for scaffolding)
